# Batch Image Crop Tool

A tool for cropping images in a batch with an interface.

## Requirement

- numpy
- opencv-python

## Usage

```bash
python main.py --input_path <YOUR_INPUT_DIR> --output_path <YOUR_OUTPUT_DIR>
```

And, it'll preview first one for cropping.
